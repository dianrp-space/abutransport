# Abu Transport - Rental Mobil Pekanbaru

![Logo Abu Transport](images/logo.webp)

> **CV. Abu Multi Jasa Trans (Abu Transport)** - Jasa rental mobil terpercaya di Pekanbaru, Riau. Menyediakan layanan lepas kunci, include driver, carteran, dropingan, travel, paket wisata, angkutan barang, dan bus pariwisata.

## 📋 Tentang Project

Website resmi Abu Transport untuk mempermudah customer dalam:
- Melihat armada yang tersedia
- Memesan rental mobil online via WhatsApp
- Mengetahui lokasi dan kontak
- Memahami layanan dan paket yang ditawarkan

## ✨ Fitur Website

- **Responsive Design** - Tampil optimal di desktop, tablet, dan mobile
- **Carousel Armada** - Showcase mobil dengan navigasi interaktif
- **Form Booking Online** - Kirim pemesanan langsung via WhatsApp
- **WhatsApp Floating Button** - Chat langsung dengan admin
- **Scroll to Top** - Navigasi cepat kembali ke atas
- **SEO Optimized** - Meta tags, Open Graph, JSON-LD Structured Data
- **Google Maps Integration** - Lokasi langsung tertanam
- **Instagram Integration** - Link langsung ke akun Instagram

## 🛠️ Teknologi

- HTML5
- CSS3 (Custom styling, no framework)
- JavaScript (Vanilla JS)
- Google Fonts (Sora & Manrope)
- WebP Images (Optimized)

## 🚀 Cara Menjalankan

1. Clone repository:
   ```bash
   git clone https://github.com/username/abutransport.git
   ```

2. Buka folder project:
   ```bash
   cd abutransport
   ```

3. Buka `index.html` di browser, atau jalankan local server:
   ```bash
   # Dengan Python
   python -m http.server 8000

   # Dengan Node.js (http-server)
   npx http-server

   # Dengan PHP
   php -S localhost:8000
   ```

4. Buka browser: `http://localhost:8000`
